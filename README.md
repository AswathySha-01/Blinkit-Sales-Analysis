# Blinkit Sales Dashboard Analytics

## Project Overview

This project is an interactive Power BI dashboard developed to analyze Blinkit's sales performance and business insights. The dashboard provides a clear view of sales trends, product performance, outlet distribution, and customer ratings using visual analytics.

## Tools & Technologies Used

- Power BI
- Microsoft Excel
- Data Visualization
- Data Analysis

## Key Features

- Total Sales Analysis
- Average Sales Analysis
- Number of Items Sold
- Average Customer Rating
- Outlet-wise Sales Performance
- Item Type Analysis
- Fat Content Analysis
- Sales Trend Visualizations
- Interactive Filters and Slicers

## Dashboard Insights

- Identifies top-performing outlet categories.
- Analyzes sales distribution across item types.
- Provides customer rating insights.
- Tracks overall business performance using KPIs.
- Helps in making data-driven decisions.

## Files Included

- Blinkit Sales Analytics.pbix
- Dataset (if included)

## Project Objective

The objective of this project is to transform raw sales data into meaningful business insights using Power BI's interactive visualization capabilities.
